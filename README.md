## ⚙️ Configuration Setup

You can adjust settings to improve visual quality and game speed.

### Graphics Settings
Open the Graphics menu within the game window. Select your display resolution to match your monitor, such as 1920x1080 or 4K. Use the texture pack toggle to enable high-resolution assets. Set the internal render scale to 2x for a balance of visual detail and system performance. Turn on V-Sync if you experience screen tearing during matches.

### Controller Setup
Connect your controller to your PC using a USB port. Navigate to the Input menu in the game options. Select your controller from the dropdown list. Map the buttons according to your preference. If you use a GameCube adapter, the software detects this automatically. Test your controls in the training mode before you start a match.

## 🌐 Playing Online

The system uses rollback netplay to reduce lag in online matches. 

1. Ensure your PC has a stable internet connection. An Ethernet cable provides better results than Wi-Fi.
2. Select the Online Play menu.
3. Choose the region closest to your physical location to keep ping times low.
4. Join an open room or create your own lobby to invite a friend.
5. Disable background tasks like video streaming or large file downloads while you play to maintain a smooth connection.

## 🛠️ Troubleshooting Issues

Use this list to solve common problems during installation or gameplay.

- The application will not open: Ensure that you extracted all files from the zip folder before running the executable. Windows may try to run the file inside the zip, which causes errors.
- Low frame rates: Open your graphics settings and reduce the internal render scale. Close other software running in the background to free up system memory and processing power.
- Controller input delay: Use a wired USB connection for your controller. Wireless Bluetooth connections can introduce slight delays during fast-paced movement.
- Sound issues: Check your Windows Sound Settings to ensure the correct output device is selected.
- Texture errors: Reinstall the texture pack by copying the files into the designated folder again. Ensure that you follow the folder structure exactly as provided in the download package.
